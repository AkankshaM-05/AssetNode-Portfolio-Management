# **App Name**: AssetNode

## Core Features:

- Wealth Intelligence Dashboard: A high-level overview featuring portfolio health indicators, total value summaries, and visual allocation cards.
- Smart Asset Registry: An interactive management interface for logging stocks and investments with local state persistence.
- AI Portfolio Strategy Tool: Uses a generative AI tool to analyze concentration and suggest diversification strategies based on mock asset data.
- Relational History Ledger: Chronological view of portfolio snapshots and analysis records prepared for PostgreSQL integration.
- Asset Filtering Engine: Dynamic search and filter system for high-volume investment tables and cards.
- Authentication Gateway: A focused, secure entry point for user sessions using standard credential validation logic.
- Relational Data Schema: Data structures optimized for eventual connection to a Cloud SQL or PostgreSQL database.

## Style Guidelines:

- Primary color: Deep Precision Indigo (#2E47B8), providing a sense of financial stability and professional trust.
- Background: Clean Mist (#F7F8FA), a very light, desaturated indigo to minimize glare and maximize readability.
- Accent: Data Blue (#36B4E5), an analogous hue used sparingly for calls to action and success states.
- Font pairing: 'Space Grotesk' for bold, tech-forward headlines and 'Inter' for clear, objective body text. 
- Utilize minimalist line-based icons with consistent stroke weights to represent asset types and menu items.
- A card-based modular grid system that prioritizes white space and information hierarchy over dense data tables.
- Functional transitions only: light fades between page states and hover-scaling for primary buttons.
