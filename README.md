# AssetNode – AI-Powered Portfolio Management Platform

## Overview

AssetNode is a full-stack portfolio management platform that enables investors to track stock holdings, visualize portfolio composition, and generate AI-powered portfolio insights.

The platform combines portfolio tracking with automated analysis to help users understand concentration risk, sector allocation, diversification, and portfolio health.

---

## Features

### Portfolio Dashboard

* Total portfolio value overview
* Active stock tracking
* Portfolio health monitoring
* Latest AI-generated portfolio assessment

### Stock Registry

* Add and manage stock holdings
* Search investments instantly
* Filter by quantity and investment cost
* Delete existing positions

### AI Portfolio Analysis

* Portfolio health scoring
* Concentration risk detection
* Sector allocation analysis
* Geographic diversification review
* Strategic portfolio recommendations

### Analysis History

* Historical AI-generated reports
* Snapshot tracking over time
* Portfolio health trend review

### Data Persistence

* Portfolio data stored in Supabase
* Historical analysis stored in database
* Real-time synchronization between frontend and backend
